# Myntra Clone UI

A responsive Myntra homepage clone built using HTML and CSS. This project recreates the look and feel of the Myntra landing page, focusing on clean layouts, modern styling, and responsive design principles.

---

## 🚀 Features

- Responsive homepage design
- Navigation bar with categories
- Banner and promotional sections
- Product showcase layouts
- Footer section
- Clean and organized HTML & CSS structure

---

## 🛠️ Technologies Used

- HTML5
- CSS3

---

## 📂 Project Structure

```
Myntra-Clone-UI/
│── images/
│── index.html
│── index.css
└── README.md
```

---

## 💻 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/sayan629/Myntra-clone-ui.git
```

2. Navigate to the project directory:

```bash
cd Myntra-clone-ui
```

3. Open `index.html` in your preferred web browser.
